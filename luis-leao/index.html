Criando experiências interativas com a Web Speech API

Por Luis Leão no DevFest São Paulo 2013

Disponibilizada em 2013 como padrão aberto no W3C, a API de reconhecimento de voz traz grandes oportunidades para criação de experiências interativas e, ao mesmo tempo, possibilidade de aplicativos e sites web focados em acessibilidade. Aprenda como implementá-la e veja exemplos de uso e boas práticas da Web Speech API.
